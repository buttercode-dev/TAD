# The Admin Department

Practical admin systems for businesses drowning in repetitive admin.

## Core promise

We turn messy admin into clear, trackable systems.

## What we are building

The Admin Department is not an AI automation agency. It is a practical back-office systems company for small businesses whose admin is stuck in WhatsApp, email, spreadsheets, documents, staff memory, and manual follow-ups.

We set up the processes, trackers, templates, reminders, reports, and approval flows a small business needs to manage repetitive admin work.

AI and automation are used behind the scenes, but important actions stay human-approved.

## MVP departments

1. **Invoice Admin** — supplier invoices, receipts, documents, approvals, filing, exceptions.
2. **Sales Admin** — enquiries, lead capture, response tracking, quote follow-up, sales reports.
3. **Client Admin** — onboarding, welcome process, document collection, folder setup, payment/request follow-up.
4. **Property Admin** — tenant messages, rent reminders, maintenance, viewings, owner reports.
5. **Practice Admin** — bookings, reminders, no-shows, receptionist admin, patient admin boundaries.
6. **Member Admin** — gym/member onboarding, attendance follow-up, failed-payment reminders, reactivation.

## First offer

**Admin Audit**

We review a business's current admin process and show where work is delayed, repeated, forgotten, or stuck in email, WhatsApp, and spreadsheets.

Deliverables:

- workflow map
- top admin leaks
- recommended department setup
- quick-win improvements
- implementation quote

The browser audit is only the diagnostic front door. A paid Admin Audit still requires a human TAD review, workflow mapping and implementation quote.

## Build rule

Do not overbuild software before a paying pilot.

First version should use simple, reliable pieces:

- Google Sheets / Supabase tracker
- Gmail/Drive intake where useful
- simple forms
- message templates
- approval queues
- dashboards/reports
- human approval before automation

The company sells organised admin outcomes, not tools.
