# PROJECT.md — The Admin Department

## Mission

Build a serious, non-sloppy admin systems company that gives small businesses the structure of an admin department without hiring a full team.

## Positioning

The Admin Department builds practical admin systems for businesses drowning in repetitive admin.

## Core promise

We turn messy admin into clear, trackable systems.

## What we do

We set up the processes, trackers, templates, reminders, reports, and approval flows a small business needs to manage repetitive admin work.

We use automation and AI where useful, but important actions stay human-approved.

## Initial market

South African SMEs and founder-led service businesses using WhatsApp, Gmail, Google Drive, spreadsheets, and manual follow-ups.

## Departments

- Sales Admin
- Client Admin
- Invoice Admin
- Property Admin
- Practice Admin
- Member Admin

## MVP scope

Build and sell:

1. Admin Audit
2. Invoice Admin Setup
3. Sales Admin Setup
4. Client Admin Setup
5. Monthly Admin Support retainer

## Out of scope for MVP

- Fully autonomous WhatsApp replies
- Autonomous payments
- Full accounting system replacement
- Medical advice or clinical triage
- Complex ERP integrations
- Multi-tenant SaaS before pilot validation

## Success metric for first 30 days

- 1 public-facing landing page
- 1 complete audit offer
- 3 department setup offers
- 100 prospect list
- 20 conversations
- 5 demos/audits
- 1 paid pilot
